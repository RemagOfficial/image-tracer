    Export panel tips:
    • Enable Convert text to paths to turn all text into SVG paths for maximum compatibility.
    • Warning: Converting text to paths may make the SVG file very large, especially for long or multi-line text.
    • To convert text, you must load the matching font file (TTF/OTF/WOFF) using the font file input. System/browser fonts cannot be converted unless you provide the font file.
    • If a font is missing, export will warn you and leave the text as <text> elements.
